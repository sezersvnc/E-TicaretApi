# E-Commerce RESTful API

A robust and scalable backend API for an e-commerce platform, built with Python and Django REST Framework (DRF). This project provides the core backend infrastructure to manage products, user authentication, shopping carts, and order processing.

## 🚀 Features

*   **User Authentication & Authorization:** Secure login and registration using JWT (JSON Web Tokens) or standard Token Authentication.
*   **Product Management:** CRUD (Create, Read, Update, Delete) operations for products, categories, and inventory management.
*   **Shopping Cart:** Add, remove, and update items in the cart.
*   **Order Processing:** Checkout functionality, order history, and status tracking.
*   **RESTful Architecture:** Clean and standardized API endpoints.
